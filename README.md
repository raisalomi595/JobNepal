# JobNepal

A job portal website built with React + Vite. Lists jobs, companies, and hiring info in a clean layout.

## Sections

- Navbar
- Hero Section (search by job title & location)
- Instant Jobs & Hot Jobs sidebar
- Top Jobs
- Featured Jobs
- Popular Categories
- Why Choose Us
- Top Hiring Companies
- Call to Action
- Footer

## How to Run

```bash
npm install
npm run dev
```

## Built With

- React
- Vite
- Tailwind CSS
